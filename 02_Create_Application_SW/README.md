# Create Application Software

---
### Launch Vitis IDE and create project workspace
```
$ source ./setenv.sh
$ cd ./appsw && \rm -rf workspace && mkdir ./workspace
$ cd ./appsw/workspace && cp -f ../../export/$(TOP).xsa .
$ cd ./appsw/workspace && vitis --workspace . &
```
